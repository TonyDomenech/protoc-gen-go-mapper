# proyecto-videojuego

Este repositorio contiene un sencillo juego de Ping Pong que puedes ejecutar directamente en tu navegador o con Python.

* **Versión HTML**: abre `html/index.html` para jugar sin instalar nada. El menú inicial permite escoger entre tres modos (Jugador vs Jugador, Jugador vs Máquina o Máquina vs Máquina) y cinco niveles de dificultad para la IA.
* **Versión Python**: instala la dependencia `pygame` con `pip install pygame` y ejecuta `python main.py` dentro de la carpeta `python`. Funciona igual que la versión de navegador y ofrece el mismo sistema de menús.

El campo es ancho para que puedas reaccionar con tiempo, la pelota espera dos segundos en el centro tras cada punto y la partida termina cuando un jugador alcanza cinco tantos. El jugador 1 es azul y el jugador 2 es rojo.
